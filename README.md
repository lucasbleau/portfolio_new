## 🎯 À Propos

Portfolio personnel moderne et responsive de **Lucas Bleau**, étudiant en Master Management et Architecture Logicielle (MAALSI) à CESI.

## ✨ Caractéristiques

- **Design moderne** avec thème sombre et effets glassmorphism

## 🚀 Lancer le Portfolio

### Option 1: Ouvrir directement
Double-cliquez sur `index.html` pour l'ouvrir dans votre navigateur.

### Option 2: Serveur local (recommandé)
Pour une expérience optimale avec un serveur local :

**Avec Python :**
```bash
# Python 3
python -m http.server 8000

# Puis ouvrez http://localhost:8000
```

**Avec Node.js (npx) :**
```bash
npx serve

# Puis ouvrez l'URL affichée
```

**Avec VS Code :**
- Installez l'extension "Live Server"
- Clic droit sur `index.html` > "Open with Live Server"
